# Computer Components Dataset
Contains CSV files with list of different components required to build a functioning personal computer (PC). Each row in CSV represents separate component of the type, and columns indicate common properties of that component such as Price, Description, Memory etc.

## Contents

- General

  - CPUs
  - CPU Coolers
  - Motherboards
  - Memory
  - Storage
  - Video Cards
  - Cases
  - Power Supplies
  - Optical Drives
  - Operating Systems
  - Monitors
  - External Storage

- Accessories / Other

  - Case Accessories
  - Case Fans
  - Fan Controllers
  - Thermal Compound
  - UPS Systems

- Expansion Cards / Networking

  - Sound Cards
  - Wired Network Adapters
  - Wireless Network Adapters

- Peripherals
  - Headphones
  - Keyboards
  - Mice
  - Speakers
  - Webcams
